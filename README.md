# 🚀 AI Career Advisor & Recommendation System

A comprehensive platform designed to help students discover their ideal career paths using Machine Learning and AI-driven counseling.

## 🌟 Key Features
- **AI Career Prediction:** Uses Machine Learning models (`.pkl`) to predict the best career based on student interests and skills.
- **AI Chatbot:** Integrated with **Groq API** to provide real-time career guidance and answer student queries.
- **College Finder:** Explore various colleges and educational opportunities.
- **Interactive UI:** A modern, responsive frontend built with React and Vite.
- **Resource Hub:** Access to entrance exam guides, skill trackers, and scholarship information.

## 🛠️ Tech Stack
- **Frontend:** React.js, Vite, CSS3
- **Backend:** Node.js, Express.js (for APIs), Flask (for ML model integration)
- **Database:** MongoDB
- **AI/ML:** Python, Scikit-learn, Groq Cloud API
- **Version Control:** Git & GitHub LFS (for large model files)
